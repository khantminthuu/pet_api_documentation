Get started

    API Endpoint

        http://pet.com//index.php/Register/loginAccount

account and password is required

Category


# Here is a curl example
curl
-X GET http://pet.com//index.php/Register/loginAccount/{account}/{password}

To get admin info you need to make a GET call to the following url :
http://pet.com//index.php/Register/loginAccount/{account}/{password} 



Result example :
http://app.pet.cn.net/index.php/Register/loginAccount?account=admin&password=123456
{
  "status": 1,
  "message": "操作成功",
  "data": {
    "token": ""
  }
}

PATH PARAMETERS

Field Type Description
account string User Account Name
password string or integer User Password

Errors

Htee API uses the following error codes:

Error Code Meaning
400 Some parameters are missing. This error appears when you don't pass username or password.
404 Unknown or unvalid user. This error appears if you use an unknow username or if the user's password.
