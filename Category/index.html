Get started

    API Endpoint

        http://pet.com//index.php/GoodsClass/getClass

fid နေရာက sidebar က . pid နေရာက navbar

Category


# Here is a curl example
curl
-X GET http://pet.com//index.php/GoodsClass/getClass/{fid}/{pid}

To get admin info you need to make a GET call to the following url :
http://ipaddress:4000/auth/login/{fid}/{pid}

Errors

Htee API uses the following error codes:

Error Code Meaning
400 Some parameters are missing. This error appears when you don't pass username or password.
404 Unknown or unvalid user. This error appears if you use an unknow username or if the user's password.
